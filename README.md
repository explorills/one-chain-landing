# ONE Chain Landing

Informational landing page for [ONE Chain](https://github.com/explorills/one-chain) — the EVM-compatible blockchain purpose-built for the [EXPL.ONE](https://expl.one) ecosystem.

## Tech Stack

- **Runtime**: Bun
- **Framework**: React 19 + TypeScript
- **Build**: Vite 7
- **Styling**: Tailwind CSS 4
- **Animations**: Framer Motion
- **Globe**: cobe (WebGL)
- **Icons**: Phosphor Icons

## Development

```bash
bun install
bun run dev
```

## Build

```bash
bun run build
```
